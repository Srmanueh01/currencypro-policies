Política de Privacidad

📱 Cambiofy - Convertidor de Divisas

Última actualización: 11 de marzo de 2026

✨ Esta aplicación NO recopila tus datos personales. Tu privacidad está protegida por diseño.

📊 1. Información que recopilamos

NINGUNA. Cambiofy no recopila, almacena, comparte ni vende ningún tipo de información personal. No solicitamos nombre, correo, ubicación ni ningún dato identificable.

📱 2. Permisos de la aplicación

La app solicita únicamente:

  • Internet: Para obtener tasas de cambio actualizadas desde APIs públicas.
  • Estado de red: Para detectar si tienes conexión y mostrar el modo offline.

No solicitamos acceso a contactos, cámara, galería, micrófono, ubicación ni archivos.

🌐 3. APIs de terceros

La aplicación obtiene tasas de cambio de las siguientes fuentes:

PRINCIPAL Fawazahmed0 Currency API
https://github.com/fawazahmed0/exchange-api

API de código abierto que recopila datos diarios de fuentes financieras públicas. Ventajas: Gratuita, sin límites de peticiones, soporta más de 150 monedas, datos históricos desde 2020, y funciona mediante archivos estáticos en CDN.

📊 Esta API proporciona datos históricos para TODAS las monedas (incluso pares exóticos) mediante tasas cruzadas calculadas a partir del dólar estadounidense.

RESPALDO ExchangeRate-API
https://www.exchangerate-api.com

Servicio de tasas de cambio en tiempo real con actualización horaria en plan gratuito. Se utiliza como fuente secundaria cuando es necesario.

HISTÓRICO Frankfurter.app
https://www.frankfurter.app

API pública de respaldo con tasas diarias del Banco Central Europeo. Proporciona datos históricos para las 33 monedas principales.

¿Cómo funcionan los datos? La API de Fawazahmed0 obtiene tasas base de fuentes oficiales y calcula tasas cruzadas para pares poco comunes (como FJD/BGN) usando el dólar estadounidense como referencia. Esto garantiza que todas las combinaciones de monedas tengan datos consistentes y precisos.

Estos servicios tienen sus propias políticas de privacidad. No compartimos información tuya con ellos, solo realizamos consultas de tasas públicas.

🎯 4. Publicidad

Cambiofy puede mostrar anuncios de Google AdMob para mantener la aplicación gratuita. Estos anuncios pueden ser personalizados según tus intereses, pero nosotros no compartimos ningún dato tuyo con Google. Google puede utilizar identificadores publicitarios para mostrar anuncios relevantes.

Para más información, consulta la Política de Privacidad de Google.

💾 5. Datos locales

La app guarda localmente en tu dispositivo:

  • Tus preferencias (moneda por defecto, modo oscuro, decimales)
  • Favoritos y conversiones recientes
  • Tasas de cambio en caché para funcionar sin conexión
  • Datos históricos de gráficos (7 días de caché)

Estos datos NUNCA salen de tu dispositivo.

👤 6. Tus derechos

Como no recopilamos datos personales, no hay información que puedas solicitar. Pero si tienes preguntas sobre la app:

  • ✅ Derecho a estar informado - Esta política te informa
  • ✅ Derecho a la privacidad por defecto - La app está diseñada sin recopilación
  • 📧 Contacto: scurrencypro@gmail.com

🔒 7. Seguridad

Al no recopilar datos, no existen riesgos de filtraciones de información personal. Todas las comunicaciones con las APIs se realizan mediante HTTPS.

📋 8. Cambios en esta política

Si en el futuro la app cambiara y empezara a recopilar datos, actualizaremos esta política y te informaremos claramente dentro de la aplicación.

🗑️ 9. Eliminación de datos

Aunque Cambiofy no recopila datos personales, los anuncios mostrados por Google AdMob pueden utilizar identificadores publicitarios anónimos. Si deseas solicitar la eliminación de los datos asociados a tu dispositivo en los sistemas de Google, puedes hacerlo de las siguientes maneras:

  • Opción 1 (recomendada): Ajustes de Google en tu dispositivo Android: Configuración → Google → Anuncios → Restablecer ID de publicidad / Eliminar ID
    Esta opción elimina el ID localmente sin necesidad de contacto.
  • Opción 2: Contactarnos directamente y trasladaremos tu solicitud a Google:
📧 Enviar solicitud de eliminación

Haz clic para abrir tu programa de correo

Una vez recibida tu solicitud, procesaremos tu petición en un plazo máximo de 30 días y contactaremos con Google AdMob para que eliminen cualquier dato asociado a tu identificador publicitario. Agradeceremos que incluyas el modelo de tu dispositivo y versión de Android para ayudarnos a identificar tus datos más rápidamente.
