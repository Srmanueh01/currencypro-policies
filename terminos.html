4. PRECISIÓN DE LOS DATOS Y FUENTES

Cambiofy obtiene las tasas de cambio de las siguientes fuentes públicas y de código abierto:

PRINCIPAL Fawazahmed0 Currency API
github.com/fawazahmed0/exchange-api

API de código abierto que recopila datos diarios de fuentes financieras públicas. Proporciona más de 150 monedas y datos históricos desde 2020.

RESPALDO ExchangeRate-API
exchangerate-api.com

Servicio de tasas en tiempo real con actualización horaria (plan gratuito).

HISTÓRICO Frankfurter.app
frankfurter.app

API pública del Banco Central Europeo con datos históricos para monedas principales.

⚠️ Importante: Las tasas de cambio pueden tener pequeñas variaciones respecto al mercado interbancario. No garantizamos la disponibilidad 24/7 de las APIs de terceros. La aplicación funciona en modo offline con datos cacheados, pero estos pueden no reflejar las tasas más recientes. No utilices esta aplicación para operaciones financieras en tiempo real o decisiones de inversión críticas.

5. GRÁFICOS Y DATOS HISTÓRICOS

La aplicación incluye gráficos de evolución de 30 días generados a partir de datos históricos reales de Fawazahmed0 API. Estos gráficos:

  • Se actualizan diariamente con nuevas tasas
  • Almacenan caché local de 7 días para reducir consultas
  • Muestran tendencias basadas en datos reales, no simulaciones
  • Pueden no incluir todos los días si la fuente no tiene datos para fechas específicas
