🔄 3. Precisión de las tasas y gráficos

Las tasas de cambio se obtienen de múltiples fuentes para garantizar la máxima precisión:

PRINCIPAL Fawazahmed0 Currency API
https://github.com/fawazahmed0/exchange-api

¿Cómo funciona? Esta API de código abierto recopila datos diarios de fuentes financieras públicas y los sirve mediante archivos JSON estáticos en CDN. Proporciona:

  • ✅ Más de 150 monedas - Incluye todas las monedas principales y exóticas
  • ✅ Datos históricos completos - Desde 2020 hasta la actualidad
  • ✅ Tasas cruzadas precisas - Para pares poco comunes (ej. FJD/BGN) calculadas mediante USD
  • ✅ Sin límites de peticiones - Al ser archivos estáticos, no hay restricciones
  • ✅ Actualización diaria - Datos frescos cada día

📈 Los gráficos de evolución de 30 días se generan con datos reales de esta API.

RESPALDO ExchangeRate-API
https://www.exchangerate-api.com

Tasas en tiempo real con actualización horaria. Utilizada como fuente secundaria para validación.

HISTÓRICO Frankfurter.app
https://www.frankfurter.app

API pública del Banco Central Europeo con datos históricos para 33 monedas principales.

📊 Sobre la precisión de los datos:

Para pares de monedas poco comunes (como Dólar de Fiyi a Lev búlgaro), las tasas se calculan mediante el método de "tasa cruzada" utilizando el dólar estadounidense como referencia. Este es el mismo método que utilizan los bancos y casas de cambio para cotizaciones de pares exóticos, por lo que los valores son matemáticamente precisos y consistentes con el mercado.

Hacemos nuestro mejor esfuerzo para que las tasas sean precisas, pero:

  • Pueden tener pequeñas variaciones respecto al mercado interbancario
  • No garantizamos disponibilidad 24/7 de las APIs
  • La app funciona offline con tasas de respaldo y datos en caché
  • Los gráficos históricos se actualizan diariamente

No uses esta aplicación para operaciones financieras críticas. Es una herramienta de referencia, no una plataforma de trading o inversión.

📊 4. Funcionalidad de gráficos

La aplicación incluye gráficos de evolución de 30 días para todos los pares de monedas. Estos gráficos se generan con:

  • Datos históricos reales de Fawazahmed0 API
  • Caché local de 7 días para reducir consultas
  • Más de 30 puntos de datos (uno por día)
  • Estadísticas de tendencia (mínimo, máximo, promedio)

Los gráficos están disponibles para TODAS las combinaciones de monedas, incluyendo pares exóticos, gracias al sistema de tasas cruzadas.
